# College Management System
CMS (College Management System) is web based application used to maintain students and teachers in college. 

### Operation & Features
There are various operation implement for both the student and teachers, Such as :
 - For Student and Teacher Basic Operation Such as Add, Get, Delete, Update.
 - Class Schedules Service : To maintain the time table for all the class in the college.
 - Attendance Service : To maintain the attendance of all the students.
 - Semester Marks Service : To maintain the semester mark of all the students.

### Frameworks, Lib and Tools Used
 - Spring boot
 - Spring Security
 - PostgreSQL Database
 - Postman

### IDE & API Testing Tool
